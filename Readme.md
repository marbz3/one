 Consta de un encriptador/Desencriptador de texto que funciona con las siguientes llaves:
 
 * a --> ai
 * e --> enter
 * i --> imes
 * o --> ober
 * u --> ufat

 opera únicamente con minúsculas y palabras sin acentos.
